This is a replica of the Instagram Web App built in Ruby on Rails. Main Features:
  * Liking Pictures
  * Users and Restrictions
  * Image Uploading
  * CRUD functionality for pictures

Gems used:
  * haml
  * simple_form
  * bootstrap
  * devise
  * paperclip
  * masonry-rails (jQuery)
  * acts_as_votable gem (upvoting images)
